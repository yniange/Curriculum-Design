学生成绩管理系统
每个学生的数据包括学号，姓名，性别，数学、物理、化学3门课
程的成绩。

系统运行时从数据文件中读取数据，系统退出时要保证当前系统数
据保存至数据文件中。系统应至少实现以下功能：
(1)添加一个学生信息
(2)删除特定学生的信息
(3)输出所有学生的成绩
(4)查询某一门课的平均成绩
(5)查询某位学生的平均成绩
(6)查询某门课的不及格人数
(7)查询平均最高分、平均最低分的学生姓名
(8)由大到小按平均成绩输出所有学生信息
条件假定：
(1)学生中无重名现象
(2)所有学生的总成绩不相等
